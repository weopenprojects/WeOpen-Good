# WeOpen-Good
## WeOpen-Good 简介 🌟
2022 年，腾源会携手开源社，与众多开源社区和开发者社区一起，重磅推出开源公益计划（WeOpen Good），秉承「聚开源智慧·行科技向善」的宗旨，希望将开源技术如人工智能、区块链、云计算、大数据、物联网等，应用在更广泛的民生领域，解决社会痛点。
WeOpen Good 开源公益计划会对征集到的、有成长性的公益优秀应用案例，给予资源支持和技术辅导，进而加速、提升开源创新在公益领域的普及与深化。

## WeOpen-Good 案例覆盖领域 📡

* 无障碍：通过技术创造更包容的解决方案，为具有不同程度生理伤残缺陷者和正常活动能力衰退者，提供良好的数字和信息服务，改善使用体验，畅享数字生活。

* 应急救灾：通过技术解决方案，提高自然灾害救助水平和突发公共事件应急能力，让数字化手段在收集传递灾情信息、搭建救助通道、推动科学决策、实施抢险措施中发挥积极作用。

* 科技助老：通过科技手段，针对银发群体面临的安全、健康、陪伴等关键痛点，在数字化适老、科技助老产品开发等方面积极探索，并提出解决方案。

* 生态环保：推动建设智慧高效的生态环境技术能力，为环境监测保护、绿色低碳发展、生物多样性等领域提供创新技术和有力支撑。

* 文化遗产保护：通过新的技术手段，创新传统文化遗产保护模式，为文化遗产的保护、开发和展现提供技术方案，推动社会公众更广泛享有人类文化遗产。

* 普惠教育或其它领域。
